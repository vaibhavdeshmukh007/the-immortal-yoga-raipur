# The Immortal Yoga Academy - Website

A premium, responsive, single-page website for **The Immortal Yoga Academy**, Raipur. Designed for wellness, speed, and seamless user experience.

## 🚀 Features
- **Mobile-First Design:** Fully responsive across all devices.
- **Modern Aesthetics:** Earthy color palette (Sage Green, Gold, Ivory).
- **Smooth Animations:** Integrated with ScrollReveal for premium feel.
- **WhatsApp Integration:** Floating button for instant booking.
- **SEO Optimized:** Meta tags, Open Graph tags, and LocalBusiness Schema.
- **Netlify Ready:** Built-in form handling and configuration.

## 📂 File Structure
```
/project-root
├── index.html        # Main structure & SEO
├── style.css         # Custom design system
├── script.js         # Interactivity & animations
├── netlify.toml      # Deployment config
└── assets

        └── instructor.png # Founder/Instructor photo
```

## 🛠️ How to Customize
1. **Change Images:**
   - Replace the files in `assets/images/` with your own photos.
   - Keep the same filenames (`hero.png`, etc.) or update the paths in `index.html`.
2. **Update Content:**
   - Open `index.html` and search for "Mr. Bhupendra Sahu" to change instructor details.
   - Update the phone number `+917441155307` in the `wa.me` links and contact section.
   - Update the email `contact@immortalyoga.in` with your actual business email.
3. **Change Colors:**
   - Open `style.css` and adjust the variables in the `:root` section.

## 🌐 Deployment to Netlify
1. **Drag and Drop:**
   - Zip the project folder.
   - Go to [Netlify Drop](https://app.netlify.com/drop).
   - Drag and drop your zip file.
2. **Git Deployment (Recommended):**
   - Push this code to a GitHub repository.
   - Connect the repository to Netlify.
   - It will automatically deploy and set up the contact form.

## 📧 Contact Form
The form is ready for Netlify. Once deployed, you will see form submissions in your Netlify dashboard under the "Forms" tab.
